# Yuki

Yuki is a PHP Manga/Manhwa/Manhua Reader 

## Installation
Requirement: Laravel 8+, PHP 7+, Composer.

```bash
git clone https://github.com/Aerysh/yuki

cd yuki

composer install

mv .env.example .env

php artisan key:generate

php artisan serve

```
Finally open http://localhost:8000/ in your web browser
